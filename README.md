# Pet_project_1
# Назва проєкту: Product and Order Analytics Dashboard  

# Мета. 
Показати ключові KPI компанії: продажі, клієнти, замовленя, виявити закономіреності у продажах, замовленнях, поведінці покупців, які корисні для підвищення бізнес ефективності.

# Дані.
Використано дані з датасету - https://www.kaggle.com/datasets/mysticfalls2903/cosmetics-sales-dataset

# Інструменти.
SQL для агрегації даних, Pandas для очищення даних, Tableau для візуалізації.

# Реалізація.

order_id

review_message
images_attached

4. Створення інтерактивного дашборд із графіками:
 - основні показники (KPI): загальні продажі, кількість замовлень, кількість клієнтів;
 - тренди: лінійний графік, що показує динаміку продажів протягом року;
 - продукти: горизонтальна діаграма з обсягами продажів для різних товарів;
 - топ‑клієнти: список найбільших покупців із сумами продажів;
 - статуси замовлень: кругова діаграма, яка показує співвідношення доставлених, скасованих та недоступних замовлень;
 - рейтинг продуктів: топ‑10 товарів за оцінками клієнтів.

Висновки по візуалізації.
По показникам - загальні продажі $ 99 862 , кількість замовлень - 258, кількість клієнтів - 9.
Тренди - на лінійному графіку видно сезонність - найбільше продажів припадає на листопад, кількість замовлень також, у грудні теж багато замовлень, але їх загальна сума менша.
Найбільше продаються товари - Fitness band, Anti-hairfall shampoo, naii polish.
Найбільший клієнт зробив замовлення на $ 17 533.
Найбільший рентинг має продукт - Body wash.
Частка доставлених замовлень - майже 95%, лише близько 4% замовлень була скасовано. 
Що можна зробити для збільшення продажів:
1. Невелика кількість клієнтів, що створює ризик втратити доходи, якщо клієнт перестане робити замовлення - тому для збільшення продажів та стабільності потрібно розширити клієнтську базу.
2. Дослідити причини скасованих та замовлень, що мають статус "unavailable" - скасування та замовлення, що не були доставлені - зменшують лояльність клієнтів, який і так не багато.
3. Перед початком сезонну інвестувати у маркетинг та запаси для продуктів із високим рейтингом, щоб збільшити продажі.
